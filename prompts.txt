# India Votes - Development Prompts

1. **Initial UI Prompt**:
   "Build a high-performance interactive educational web application for the Indian election system using React and Vite. Use a glassmorphic design system with vibrant dark and light modes. Include a multi-step timeline of the election process."

2. **Accessibility & Quality Refactor Prompt**:
   "Refactor the codebase to achieve a 100% accessibility score. Use semantic HTML5 tags (<main>, <section>, etc.). Add aria-label to all interactive elements and descriptive alt text to all images. Modularize the App.tsx by extracting components into a separate directory with strict TypeScript interfaces."

3. **Gemini & Wikipedia Integration Prompt**:
   "Integrate the Google Gemini 1.5 Flash API to summarize Wikipedia extracts for a voter-friendly experience. Implement a floating chatbot that can fetch data from Wikipedia and use Gemini for intelligent summarization. Use React.lazy for heavy modals."

4. **Testing Prompt**:
   "Configure Vitest and React Testing Library. Generate App.test.tsx to ensure all core components (Hero, Flashcards, Forms) render correctly. Add a 'test' script to package.json."

5. **Final Efficiency & Google Services Prompt**:
   "Refactor the app for maximum efficiency using React.lazy and Suspense for all major components. Integrate Google Firebase Analytics for user tracking. Add JSDoc documentation to all utility functions and ensure zero 'any' types in the TypeScript codebase."

6. **Deployment Prompt**:
   "Dockerize the application using Nginx on port 8080 and Node 20. Deploy to Google Cloud Run under project 'promptwars-virtual-494408' with service name 'civic-navigator'."
