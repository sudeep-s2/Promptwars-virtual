# India Votes - Project Requirements

1. **Accessibility (ARIA)**: 
   - [x] All buttons must have 'aria-label'.
   - [x] All images must have descriptive 'alt' text.
   - [x] Use semantic HTML tags (<main>, <header>, <section>, <footer>).
   - [x] 100% score on accessibility audits.

2. **Security & Validation**:
   - [x] Strict EPIC Regex: ^[A-Z]{3}[0-9]{7}$.
   - [x] Input sanitization for all forms.
   - [x] No dangerouslySetInnerHTML for untrusted content.

3. **Performance (Maximized)**:
   - [x] Component extraction & modularity.
   - [x] 100% Lazy loading for all major components.
   - [x] Optimized asset delivery.

4. **Testing**:
   - [x] Unit tests for all core UI flows using Vitest.
   - [x] 100% component rendering test coverage.

5. **AI & Integrations**:
   - [x] Google Gemini 1.5 Flash for summarization.
   - [x] Real-time Wikipedia API integration.
   - [x] Google Firebase Analytics for user tracking.

6. **Code Quality**:
   - [x] Strict JSDoc documentation for all utility functions.
   - [x] Zero 'any' types in the TypeScript codebase.

7. **Deployment**:
   - [x] Dockerized environment.
   - [x] Continuous deployment to Google Cloud Run (Service: civic-navigator).
