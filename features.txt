# India Votes - Interactive Election Assistant

## Features Overview

1. **Vibrant Dark & Light Mode Toggle**
   - Seamlessly switch between a sleek deep-space dark theme and a vibrant bright blue/green light theme. Uses CSS custom properties and glassmorphic designs to maximize aesthetic appeal.

2. **Interactive Election Timeline**
   - Step-by-step guide to the election process, from constituency delimitation to the formation of the government, including the cVIGIL citizen vigilance app.

3. **Election Types (PM vs State)**
   - Explains the critical differences between General (Lok Sabha) and State (Vidhan Sabha) elections.

4. **Dynamic Wikipedia Integration**
   - **Election Dates Component:** Allows users to check details for PM or State Assembly elections. It incorporates a strict failsafe mechanism requiring users to enter exact valid Indian State names, then dynamically fetches full-text extracts directly from the Wikipedia API.
   - **Informative Chatbot:** A floating AI chatbot that recognizes common abbreviations (EVM, VVPAT, NOTA, ECI). For unrecognized queries, it searches Wikipedia, provides a snippet, and generates a "More details" button which opens a beautiful full-screen modal containing the complete text.

5. **Flashcards & FAQs**
   - A symmetrical (3x2) set of 3D flipping flashcards explaining crucial election components, forms (Form 6, 7, 8), and the Model Code of Conduct (MCC).

6. **Candidate KYC Search**
   - A filterable interface to search candidates by their constituency to ensure informed voting (includes mock criminal background data).

7. **Simulated Utilities & Forms**
   - **EPIC Validator:** A regex-powered tool to check if a Voter ID matches the standard format (e.g., ABC1234567).
   - **Voter Registration (Form 6):** An expandable mock form capturing user details (Name, Age, Nationality, Address, ID Proof). Includes strict regex and numerical boundary validations (e.g., rejecting alphabetical ages and numerical names).
   - **Queries Form:** An expandable form for users to contact support, also equipped with comprehensive validation checks.

## Technology Stack
- **React & Vite**: Extremely fast development build and rendering.
- **Framer Motion**: For fluid animations, layout transitions, 3D card flips, and full-screen modals.
- **Lucide React**: Scalable SVG icons.
- **Vanilla CSS**: Optimized with CSS Custom Properties for dynamic theme switching and `will-change` hints for low-RAM performance. No external CSS frameworks were used.
