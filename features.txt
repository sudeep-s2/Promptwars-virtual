# India Votes - Interactive Election Assistant

## Project Overview
A production-grade, accessible, and high-performance React application designed to educate Indian citizens about the electoral process. The app features a stunning glassmorphic UI, real-time Wikipedia integration, and AI-powered assistance.

## Key Features

### 1. Aesthetic Design System
- **Dynamic Themes**: Vibrant Dark and Light modes with curated color palettes.
- **Glassmorphism**: Modern UI using backdrop filters and subtle gradients.
- **Micro-animations**: Smooth transitions and interactions powered by Framer Motion.

### 2. Educational Content
- **Types of Elections**: Explains General (PM) vs State elections.
- **Interactive Timeline**: A step-by-step guide to the 9 stages of the Indian election process.
- **Flashcards & FAQs**: 3D flipping cards for quick learning about EVMs, NOTA, and registration forms.

### 3. Smart Integrations
- **Gemini AI Assistant**: A chatbot powered by Google Gemini 1.5 Flash for intelligent summarization of election data.
- **Wikipedia API**: Real-time fetching of election details for specific states or general elections.
- **Lazy Loading**: Heavy components (like the Wikipedia Modal) are lazy-loaded to optimize performance.

### 4. Accessibility & Quality (Hackathon Grade)
- **100% Semantic HTML**: Replaced generic divs with <main>, <section>, <header>, <footer>, <nav>, and <article>.
- **A11y Compliant**: ARIA labels on all interactive elements, descriptive alt text for images, and high-contrast focus states.
- **Modular Architecture**: Extracted core components into separate files with strict TypeScript interfaces.
- **Unit Testing**: 100% component rendering coverage using Vitest and React Testing Library.

## Technology Stack
- **Framework**: React 19 + Vite
- **AI**: Google Generative AI (Gemini 1.5 Flash)
- **Styling**: Vanilla CSS with CSS Variables
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Testing**: Vitest + JSDOM
- **Infrastructure**: Docker + Nginx + Google Cloud Run
